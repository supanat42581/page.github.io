CSS syntax ประกอบด้วยอะไรบ้าง
selector, property, value
การเขียน CSS มีกี่แบบอะไรบ้าง
3แบบ Inline, Internal, External
selector มีกี่แบบอะไรบ้าง
id, class, combinators, Pseudo, Pseudo class
combinator มีกี่แบบอะไรบ้าง
  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)
property และ value คืออะไรแตกต่างกันอย่างไร
property คือสิ่งที่เราจะทำการตกแต่ง ส่วนvalue คือ ค่าที่เรากำหนดเข้าไป
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
  • Content เช่น font-size, font-family
  • Border เช่น border-style, border-color
  • Padding เช่น padding-left padding-right padding-top padding-bottom
  • Margin เช่น margin-left margin-right margin-top margin-bottom
  • Element เช่น height,weight,flex,grid
value unit มีกี่ประเภทอะไรบ้างแต่ละประเภทคืออะไร
  • Absolute พวก px
  • Relative พวก em rm %
